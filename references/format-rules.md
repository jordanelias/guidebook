## Heading Hierarchy
- H1 `#` — Volume title only
- H2 `##` — Part / major division
- H3 `###` — Category / chapter
- H4 `####` — Section
- H5 `#####` — Item (specification item)

## Item Code Format
- Canonical: V{vol}-P{part}-{XX} (e.g. V2-P4-01)
- Never: V2-PV-XX or any variant

## Table Format
- Pipe tables only. No grid tables.
- Header separator: `|---|---|`
- No cell wrapping across lines

## Timestamp Format
- All timestamps: YYYY-MM-DD HH:MM (no date-only entries)

## Citation Format
- Author (Year). Title. Journal, Vol(Issue), pp. DOI
- Unverified: append [UNVERIFIED — DOI required]
