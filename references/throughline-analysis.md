# Throughlines and Meta-Throughlines Across Evidence Bases

**Generated:** 2026-04-24
**Model:** Opus 4.6
**Scope:** All evidence bases — 78+ BPCs, 189 connections, 30 FDR files, 557 verified bibliography entries, 90 search-logs with 46-jurisdiction tracking, opus divergence synthesis (13 parameters), item-jurisdiction divergence matrix (91 items), Phase 2A jurisdiction audit, project standards, Part 1 foundations

## Part I: Throughlines

### T-01. Code-as-Floor / Beyond-Code

The most pervasive throughline. Every evidence base independently confirms that statutory building codes are insufficient for genuine accessibility.

**Where it appears:**
- Part 1 §1.2 — explicit doctrinal statement
- Opus divergence synthesis — US ADA identified as having the most significant gaps (no LRV, no rest seating, no melanopic EDI, no pattern avoidance)
- Jurisdiction sweep — 25 of 91 Part 4 items (27%) have NO equivalent in ANY jurisdiction's code
- Search-logs — PAIN and OFS built environment needs are addressed by zero codes worldwide
- Sensory rooms (A-16) — no code mandates them; PAS 6463 is closest and only advisory
- Circadian lighting (B-01) — no national code mandates melanopic EDI
- Connection register — CON-0090 (Carnemolla 2025) directly documents that code-compliant bathrooms still fail wheelchair users

**Implication for the guidebook:** This throughline is not an argument to be made — it is a structural reality the evidence base has made irrefutable. The guidebook's 27% original-contribution items are its primary value proposition.

---

### T-02. Convergent Specification from Independent Clinical Mechanisms

Multiple populations arrive at the same built environment specification via entirely different clinical pathways, without cross-referencing each other in the literature.

**Where it appears:**
- CON-0186 — sensory relief space: NDV (sensory overwhelm), MH (hypervigilance/PTSD), DEM (agitation management), OFS (post-exertional malaise) all require a low-stimulation retreat room, but each for different neurological/psychological reasons
- CON-0017 — user control over environment: documented as highest-impact provision across NDV, MH, OFS, DEM, NEU, DEAF, VIS — seven populations, seven independent evidence streams
- CON-0183 — RT60 ≤0.3s: DEAF (speech perception), NDV (sensory processing), MH (hypervigilance) all require the same reverberation target
- CON-0182 — circadian lighting: DEM (sundowning), MH (mood regulation), NEU (fatigue cycles), OFS (circadian disruption) all converge on the same melanopic EDI specifications
- Rest seating interval — OFS (fatigue), PAIN (pain-limited walking), MOB (reduced endurance), DEM (disorientation) all benefit from ≤20m seating, each for distinct functional reasons

**Implication for the guidebook:** Convergent specification from independent mechanisms is epistemically stronger than any single-population evidence claim. These convergences are the strongest candidates for elevation to Universal Mode universal provisions. The guidebook has not yet systematically exploited this — most convergences are documented in the connection register but not yet surfaced in Part 3 as a methodological principle.

---

### T-03. Harm Asymmetry as Conflict Resolution

When two populations require opposing environmental parameters, the population experiencing greater harm from the wrong default sets the default, with supplementary individual provision for the other.

**Where it appears:**
- Project standards — explicit rule: "identify which population experiences greater harm from the wrong default; set default to protect the higher-harm population"
- Thermal conflict (F-07) — FMS cold pain (reversible) vs MS/SCI neurological deterioration (irreversible): cool default 18–20°C + local warmth for FMS
- LRV contrast (C-04) — VIS detection need (≥70) vs DEM void misperception (<50): compromise at ≥50 decision points, ≥30 floor junctions, ≥70 only at stair nosings
- Visual alarm (B-10) — DEAF need for visual alert vs NEU seizure risk from flash: ≤2 Hz resolves the conflict by staying below seizure-trigger threshold
- Opus divergence synthesis §5 — LRV cross-population conflict documented as the exemplar

**Implication for the guidebook:** Harm asymmetry is currently stated as a project standard and applied in specific conflict resolutions, but Part 3 §3.8 does not yet formalise it as a named decision principle with a severity taxonomy. "Irreversible > reversible > discomfort" is the implicit hierarchy. Making this explicit would strengthen Part 5 and reduce the need for case-by-case Opus adjudication.

---

### T-04. The "Adjustable" Operability Problem

"Adjustable" is used across evidence bases as a conflict resolution, but adjustability is only accessible to populations who can operate the adjustment mechanism.

**Where it appears:**
- Project standards — "Adjustable is not a universal conflict resolution. Document operability per population. Flag populations who cannot self-adjust."
- CON-0017 — H-02 (Individual Environmental Control) proposed as Universal Mode, but requires per-population operability assessment
- FDR-NEW-15 — tremor (PD/MS): touchscreen inaccessible, toggle/rocker preferred
- FDR-NEW-16 — tetraplegia (C4-C5): no hand function, requires sip-and-puff/eye-gaze/voice
- Kitchen worktop (I-02) — DE mandates height-adjustable, but electric height-adjustment requires fine motor to operate controls
- CON-0121 — circadian lighting user control resolves VIS photophobia conflict, but assumes user can reach and operate dimmer

**Implication for the guidebook:** Every specification using "adjustable", "individual control", or "user-operated" needs a three-part population operability note: (1) who can self-adjust, (2) who requires carer/staff operation, (3) default setting when no individual is present with harm-asymmetry rationale. This rule exists in project standards but has not been systematically applied across all Part 4 items.

### T-05. Temporal / Fluctuating Capacity

Codes assume fixed functional states. The evidence base documents that functional capacity fluctuates within days, across seasons, and over disease progression.

**Where it appears:**
- Part 1 §1.3 — "MS, PAIN, and CFS/ME conditions produce good days and bad days. PCS produces hour-to-hour variation."
- FDR-NEW-09 — OFS bed rising: staged rising protocol for variable orthostatic tolerance
- Thermoregulation — MS Uhthoff's phenomenon (heat-triggered neurological deterioration) is episodic, not constant
- Rest seating — OFS fatigue onset distance varies by day (15–30m)
- DAR framework — entire DAR concept addresses temporal decline: build provisions now for future capacity change

**Implication for the guidebook:** Part 1 states this clearly but the specification language in Part 4 does not consistently accommodate it. Specifications written as fixed thresholds ("≤20m rest seating interval") implicitly assume consistent capacity. A fluctuation note should accompany specifications where the target population has documented episodic variation.

---

### T-06. Compound Functioning / Supra-Additive Disability

Co-occurring conditions produce effects that exceed the sum of individual impairments.

**Where it appears:**
- Framework compound functioning review — Clarke: 4.52× environment-impairment interaction; MWI: ~16% per additional condition
- FDR registry §C — 7 compound scenarios (DEM+VIS wayfinding, MOB+PAIN grab bar, DEM+DEAF communication, etc.)
- CON-0187 — FDR-CMP-04 sequential wayfinding: DEM+VIS+IntD compound is qualitatively different from any pairwise combination
- CON-0182 — circadian lighting compound: DEM+MH vs NDV+OFS produce different interaction effects on the same lighting specification
- Project standards — "Pairwise population conflict resolution is necessary but insufficient for intra-individual compound profiles. Compound profiles route to Mode S OT co-design."

**Implication for the guidebook:** The pairwise conflict resolution framework in Part 3 §3.8 and Part 5 handles inter-population conflicts (different people in the same space). Compound functioning (multiple conditions in the same person) is acknowledged but not yet operationalised beyond "route to Tier 2." The FDR compound scenarios provide the evidence basis for a more structured compound assessment protocol.

---

### T-07. Specification Ranges as Tier 1→Tier 2 Bridges

Ranges in specifications are not expressions of uncertainty — they are the designed-in pathway from population-level defaults to individual co-design.

**Where it appears:**
- Project standards — explicit doctrine
- Opus divergence synthesis §4 (reach range) — 900-1100mm functional target with "Mode P default: median. Tier 2: OT assessment resolves position within range"
- Opus §9 (grab bar height) — no single height optimal; composite specification (700-800mm horizontal + 900mm vertical) needed because optimal height depends on transfer type
- Turning space (E-01) — graduated values by device type (1500mm manual, 1700mm powered, 2100mm complex)

**Implication for the guidebook:** This principle is stated in project standards and applied in the opus synthesis. The Part 4 format rule (every range carries a Mode S handoff flag) exists but has not been systematically applied across all items.

---

### T-08. Standards Archaeology

Current code values embed clinical assumptions from the 1960s-70s that were never revisited with modern evidence or modern mobility devices.

**Where it appears:**
- Opus §1 (ramp gradient) — 1:12 "set in the 1960s-70s based on able-bodied testing with manual wheelchairs"
- Opus §4 (reach range) — US 380-1220mm "designed for the 5th-95th percentile of wheelchair users reaching from a stationary position" — does not account for UPL, OFS, or powered chairs
- Opus §6 (turning space) — 1500mm "derived from manual wheelchair dimensions (max 750mm × 1200mm)" — modern powered wheelchairs measure up to 800mm × 1400mm
- Opus §3 (corridor width) — US 915mm "set for manual wheelchair passage only"
- Opus §9 (grab bar height) — US 838-914mm "reflects an ambulant-user bias in the ADA specification"
- Cross-cutting observation §3 — "The ADA was written in 1990 and has not been substantially updated for 35+ years"

**Implication for the guidebook:** This throughline is the historical justification for the guidebook's existence. Codes were calibrated to a population that no longer represents modern disabled building users (powered wheelchair users, people with complex neurological conditions, neurodivergent populations). Part 1 should explicitly name this as the systemic failure the guidebook addresses.

---

### T-09. Test Method Divergence as Invisible Barrier

Specifying a numerical target without specifying the test method is meaningless when jurisdictions use incompatible methods for the same property.

**Where it appears:**
- Opus §10 (slip resistance) — 4 incompatible test methods (DCOF, PTV, R-value, ramp+pendulum); a floor passing US DCOF ≥0.42 may fail UK PTV ≥36
- Opus cross-cutting §2 — "STC vs Rw vs DnT,w" for acoustic transmission
- Project standards — "STI is correct criterion [for acoustic performance], not NRC"
- Jurisdiction divergence matrix — TWSI systems physically different across JP/ISO/DE/AU/US

**Implication for the guidebook:** Part 4 must specify both the target value AND the test method for every quantified performance criterion. Part 8 (Engineering) should contain a test-method concordance table for the major parameters. This is a distinct publication-grade contribution the guidebook can make.

---

### T-10. Global South Evidence Gap / Formal-Informal Divide

The evidence base is structurally concentrated in 6 jurisdictions (US, UK, AU, CA, DE, ISO) with near-total absence from 24 Global South jurisdictions.

**Where it appears:**
- Jurisdiction audit — 24 ABSENT jurisdictions (all Global South + INT/UN)
- Project standards — "formal/informal city divide is the meta-barrier to accessible housing globally"
- Search-logs — 11 THIN jurisdictions (BD, EG, PE, GT, EC, UY, KE, TH, MA, GH, ET)
- Kawa model — PAS 6463 private retreat spatial model does not transfer to collectivist cultures
- Project standards — "ISO wayfinding specifications are physically determined and transfer universally without cultural or climate adaptation. The Global South wayfinding gap is implementation and maintenance, not specification design."

**Implication for the guidebook:** The guidebook specifies for formal new construction and planned retrofit only (stated in project standards). Informal settlement housing is out of scope. Part 1 must name this boundary explicitly and identify it as the highest-priority research and policy gap. The physics/spatial-model split (T-11 below) determines which specifications can be applied across cultures and which require local adaptation.

---

### T-11. Physics Transfers Universally; Spatial Models Don't

Measurable environmental parameters (acoustic, lighting, thermal, contrast) are universal. How people use space is culturally contingent.

**Where it appears:**
- Project standards — PAS 6463 sensory principles split: "physics-based (acoustic, lighting, visual complexity) transfer universally; spatial model (private individual retreat) does not transfer to collectivist cultures"
- Kawa model — informs non-Western sensory space specification; designs for "shared calm (collective flow) rather than individual withdrawal"
- Global South sensory space research — emerging evidence; no alternative spatial model documented in literature
- Thermoregulation — 18–22°C working target is clinically determined (physics); room zoning patterns are culturally variable
- ISO wayfinding (23599, 21542) — "physically determined and transfer universally"

**Implication for the guidebook:** Every specification should be examinable through this lens: is this a physics parameter or a spatial model? Physics parameters go into Universal Mode/1 with universal application. Spatial models go into Tier 1 with a Tier 2 cultural adaptation flag. The guidebook does not yet apply this systematically.

## Part II: Meta-Throughlines

Meta-throughlines are higher-order patterns that emerge from the interaction of multiple throughlines.

---

### M-01. The Environment Creates Disability

**Constituent throughlines:** T-01 (code-as-floor), T-08 (standards archaeology), T-05 (fluctuating capacity), T-10 (Global South gap)

Lawton's docility-proactivity continuum (CON-0028), the ICF model, the Part 1 definition of disability ("the mismatch between a person's impairment and an environment that has not been designed to accommodate it"), and the CRPD Art. 9 obligation all converge on the same proposition: disability is a property of the person-environment interaction, not of the person. The guidebook operationalises this through specifications that modify the environment rather than requiring the person to adapt.

Every throughline derives from this meta-throughline. Code-as-floor (T-01) exists because codes set the environmental modification threshold too low. Standards archaeology (T-08) exists because codes were calibrated to a narrow population model. Fluctuating capacity (T-05) exists because codes assume a stable person interacting with a fixed environment. The Global South gap (T-10) exists because the enabling environment has not been built at all.

**Structural role in the guidebook:** This is the intellectual foundation. Part 1 §1.1 states it. The three-tier design hierarchy operationalises it (Universal Mode = anti-docility floor, Tier 1 = population-calibrated press reduction, Tier 2 = proactive person-environment optimisation). The four-framework layering (ICF + PEO/PEOP + Capability Approach + Kawa) provides the theoretical scaffolding.

---

### M-02. Convergence as Epistemological Warrant

**Constituent throughlines:** T-02 (convergent specification), T-03 (harm asymmetry), T-06 (compound functioning)

When independent populations arrive at the same specification via independent clinical mechanisms, the resulting specification has a qualitatively different epistemological status from a single-population finding. It is not merely "supported by more evidence" — it is supported by independent evidence, which is the strongest form of scientific corroboration.

The guidebook has not yet articulated this as a methodological principle. The connection register documents the convergences (CON-0186 is the strongest example: four populations, four mechanisms, one room type). The opus divergence synthesis documents cross-jurisdictional convergences. But Part 3 does not state the epistemological principle: "specifications supported by convergent evidence from independent populations are stronger than specifications supported by any single population's evidence base, regardless of the quality of that single base."

This meta-throughline has practical consequences:
- Convergent specifications should be identified, labelled, and elevated in the evidence hierarchy
- They are the strongest candidates for Universal Mode universal provisions
- They resolve the "thin evidence base" problem for populations where single-population evidence is limited (PAIN, OFS) — if the specification converges with NDV and DEM evidence, the thin individual base is compensated by convergent confirmation

---

### M-03. The Precision-Universality Tension

**Constituent throughlines:** T-04 (adjustable operability), T-05 (fluctuating capacity), T-06 (compound functioning), T-07 (ranges as bridges)

The guidebook must be simultaneously universal (Universal Mode/1 population-level specifications that any architect can apply without OT involvement) and precise (Tier 2 individual co-design that accommodates the specific person). Every throughline contains this tension:

- Grab bars: a universal height is clinically impossible → composite specification (horizontal + vertical) resolves at Tier 1; OT clinical positioning resolves at Tier 2
- Thermal environment: a universal temperature is physiologically impossible → cool default via harm asymmetry at Tier 1; individual local warmth provision at Tier 2
- LRV contrast: a universal contrast level creates cross-population conflict → graduated thresholds by location type at Tier 1; individual assessment at Tier 2
- User control: "adjustable" is universally desirable but not universally operable → population operability notes at Tier 1; individual interface specification at Tier 2

The resolution mechanism is always the same structure: Tier 1 defaults via harm asymmetry or convergent evidence → Tier 2 resolution via OT assessment within the range. Specification ranges are the designed-in bridge. The guidebook equips professionals with decision frameworks; it does not prescribe for individuals.

This meta-throughline is the central design philosophy of the guidebook. It could be stated as: **"Universal provisions create the floor; individual provisions optimise within the range the floor establishes."**

---

### M-04. Systematic Absence as Systematic Evidence

**Constituent throughlines:** T-01 (code-as-floor), T-10 (Global South gap), T-08 (standards archaeology)

The jurisdiction sweep systematically documented not only what codes say but what they do not say. These absences form a coherent pattern:

- **Population absences:** No code worldwide addresses PAIN or OFS built environment needs. No code mandates sensory rooms. No code specifies melanopic EDI. These are not random gaps — they reflect populations whose access needs were not conceptualised when codes were written.
- **Parameter absences:** US ADA has no LRV, no rest seating, no hearing loop performance standard (until A117.1-2017). These are not minor omissions — they are entire sensory/cognitive access domains left unspecified.
- **Jurisdiction absences:** 24 Global South jurisdictions have zero BPC confirming evidence. Legislation exists in most (CRPD ratified by 45/46) but specification does not follow.

The systematic documentation of these absences across 46 jurisdictions × 91 items is itself a publication-grade contribution. It transforms anecdotal claims ("codes are inadequate") into quantified evidence ("27% of specifications have no code equivalent in any of 46 jurisdictions").

---

### M-05. The Two Transfers

**Constituent throughlines:** T-09 (test method divergence), T-11 (physics vs spatial models), T-10 (Global South gap)

There are two kinds of cross-jurisdictional transfer, and they operate by entirely different mechanisms:

**Physics transfer:** Acoustic reverberation, luminance contrast, thermal thresholds, slip resistance, threshold height — these are determined by human physiology interacting with measurable environmental parameters. A ≤5mm threshold is imperceptible to all users regardless of culture. RT60 ≤0.3s improves speech perception for hearing-aid users in any language. These transfer universally. The only barrier to transfer is test method incompatibility (T-09), which is a standardisation problem, not a specification problem.

**Spatial-model transfer:** How people use retreat spaces, how families configure domestic space, how wayfinding assumptions map to cultural navigation models — these are culturally contingent. PAS 6463's private individual retreat model is the product of an Anglo-Northern European individualist spatial culture. It does not transfer to collectivist cultures (Japan, India, West Africa). The Kawa model (Iwama 2006) provides the theoretical bridge but no alternative spatial specification yet exists.

The guidebook's approach: specify physics universally; flag spatial models for Tier 2 co-design with cultural context input. This is stated in project standards but not yet operationalised as a systematic classification across all Part 4 items.

---

### M-06. The Guidebook as Standards Gap-Filler

**Constituent throughlines:** T-01, T-02, T-08, T-10, M-04

The final meta-throughline synthesises the others into a statement of purpose. The guidebook exists because:

1. Codes were written for a narrow population (T-08) and have not been updated (T-01)
2. 27% of the specifications needed for genuine accessibility have no code equivalent anywhere (M-04)
3. The populations with the thinnest code coverage (PAIN, OFS, NDV, MH) are the fastest-growing disability categories in ageing societies
4. The evidence for cross-population convergence (T-02) is stronger than any single code's evidence base but has never been synthesised into a unified specification
5. The Global South has legislation without specification (T-10) — the guidebook provides the specification

The guidebook is not an alternative to codes. It is not a wish list. It is the specifications that codes have not yet produced, derived from evidence that codes have not yet incorporated, for populations that codes have not yet recognised. Its primary contribution is the 27% of items that exist nowhere else.

## Part III: Structural Implications

### For Part 1 (Foundations)
- Name the convergence-as-epistemological-warrant principle (M-02)
- Name the precision-universality resolution mechanism (M-03) — "Universal provisions create the floor; individual provisions optimise within the range"
- Name the two-transfer distinction (M-05) — physics vs spatial models
- Quantify the systematic absence finding: "27% of items, zero codes, 46 jurisdictions surveyed" (M-04)

### For Part 3 (Synthesis Framework)
- Add convergent-specification identification as a formal step in §3.8 decision tree
- Formalise harm-asymmetry severity taxonomy: irreversible neurological > reversible physiological > discomfort
- Add compound-functioning screening as Step 0 (currently stated in project standards but not formalised in Part 3 text)
- Add physics/spatial-model classification as a specification attribute

### For Part 4 (Item Specifications)
- Systematic application of population operability notes to all "adjustable" items
- Systematic application of Mode S handoff flags to all range specifications
- Convergence labels where 3+ populations independently require the same specification
- Test-method specification for all quantified performance criteria

### For Part 5 (Co-Occurrence Resolution)
- Formalise harm-asymmetry as a named decision principle
- Distinguish inter-population conflict (different people, same space) from compound functioning (same person, multiple conditions)
- Reference convergent specifications as pre-resolved: where populations converge, no conflict exists
