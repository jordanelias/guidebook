## FDR-COGWAY-01 — d460 + MCI / dementia → wayfinding (decision architecture, signage placement)

**Sources:** PMC8725382 (wayfinding design criteria SR, Tier 2); Pagethink inclusive wayfinding design; OT clinical reasoning

| Parameter | Value | Condition | Source | Tier | Delta |
|---|---|---|---|---|---|
| Information at decision point — not ahead | Signage positioned at or just before decision point; avoid placing signs 5–10m before decision point | MCI/cognitive impairment: information presented before a decision point is not held in working memory until the decision is reached — sign must be at the decision point itself | Pagethink inclusive design; OT clinical reasoning | Co-2 | **NOVEL** — decision-point-proximate signage placement as cognitive wayfinding specification absent from BPC |
| Signage — single message per sign | One direction per sign; avoid compound signs listing multiple destinations | Cognitive impairment: multiple-destination signs require selection, which is a complex cognitive task — single direction signs eliminate the selection step | PMC8725382; OT clinical reasoning | Tier 2/Co-2 | **NOVEL** — single-message sign constraint as cognitive wayfinding specification absent from BPC |
| Landmark differentiation at decision points | Unique, salient feature (colour, object, texture change) at every route decision point | MCI/dementia: homogeneous intersections are disorienting; unique landmarks at each decision point provide both current position cue and memory anchor for return navigation | PMC8725382 | Tier 2 | REFINES FDR-DEM-01 finding; extends to general cognitive wayfinding |
| Visual confirmation of destination — approach | Visible element of destination (colour, object, signage) visible from ≥5m approach | Cognitive impairment: destination confirmation before arrival reduces path-abandonment; user must be able to recognise they are on the correct route before arriving | PMC8725382; PMC8545728 | Tier 2 | REFINES FDR-DEM-01 (destination visibility); confirms as general cognitive principle |

## FDR-COGWAY-02 — d460 + ID → wayfinding (same findings cross-reference)

**Cross-reference:** FDR-ID-01 findings apply directly. Key additions specific to cognitive wayfinding context:

| Parameter | Value | Condition | Source | Tier | Delta |
|---|---|---|---|---|---|
| Route simplicity — no backtracking | Design routes that do not require retracing the same path (one-way loops preferred in complex buildings) | ID + MCI: backtracking requires recollection of the outward journey — loop routes eliminate this memory demand | OT clinical reasoning; PMC8725382 | Tier 2/Co-2 | **NOVEL** — one-way loop route as cognitive wayfinding provision absent from BPC |
| Consistent landmark persistence | Landmarks used as route anchors must be permanent — avoid seasonal changes, rotating displays, or removable elements | MCI/dementia: learned cognitive maps incorporate specific landmarks; removal or change of a landmark invalidates the learned map | PMC8545728 | Tier 2 | **NOVEL** — landmark permanence as cognitive wayfinding specification absent from BPC |
