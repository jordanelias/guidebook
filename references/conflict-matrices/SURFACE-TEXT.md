## Conflict Domain: SURFACE-TEXT
**Parameter:** Floor/path surface texture
**Status:** CONVERGENT — Opus synthesis 2026-03-30; Sonnet writeup 2026-03-30 22:36
**Date created:** 2026-03-30
**Feeds:** Part 5 §5.2 conflict resolution table

### Opposition summary
| Pop A need | Pop B need | Classification |
|---|---|---|
| VIS, DBL: tactile differentiation, detectable warnings (TWSI raised profile ≥4 mm, ISO 23599:2019) | MOB: smooth traversal, minimal rolling resistance | CONVERGENT (pre-resolved) |

### Resolution

**Classification: CONVERGENT.** The apparent conflict between tactile surface indicators (VIS/DBL) and smooth traversal (MOB) is resolved by truncated dome sizing per ISO 23599:2019 and JIS T 9251:2006. Dome dimensions (5 mm height, 25 mm diameter, 50–75 mm spacing) are detectable by white cane and foot but traversable by wheelchair with minimal rolling resistance increase. This resolution is the most mature in the compendium — embedded in international standards with 14-language jurisdictional adoption.

**Key evidence:**
- ISO 23599:2019: TWSI profile ≥4–5 mm; dome spacing designed to balance detectability and traversability. Tier 4, adopted in all 24+ jurisdictions.
- JIS T 9251:2006 (JA): 点字ブロック specifications; dome geometry optimised for both cane detection and wheelchair passage. Tier 6.
- GB 50763-2012 §3.2 (ZH): 盲道 specifications with raised profile. Tier 6.
- ABNT NBR 16537:2024 (PT): extends tactile paving guidance explicitly to surdo-cegueira (DBL). Tier 4.
- DIN 32984:2011-10 (DE): Bodenindikatoren im öffentlichen Raum. Tier 5/6.
- VIS BPC Opus synthesis: TWSI/DEM conflict (yellow paving vs plain flooring) resolved via profile-based detection over colour-based. E-09 conflict note.
- MOB BPC Opus synthesis: "No cross-population conflict identified for MOB provisions. MOB turning circles, ramp gradients, and door widths are beneficial or neutral for all other populations. MOB is the only population code with zero conflict flags."
- DBL BPC Opus synthesis: vibrotactile floor panels vs TWSI resolved (TWSI takes precedence; vibrotactile via wearable pager).

### Tier 0 specification
Single Tier 0 value viable: ISO 23599:2019 TWSI (directional ribbed + attention domed) on all primary circulation routes. Profile ≥4 mm. No population conflict.

### Secondary conflict: TWSI colour vs DEM plain flooring
**Populations:** VIS (yellow TWSI standard) vs DEM (plain, low-pattern flooring)
**Resolution:** RESOLVED-CONSENSUS. Profile-based detection is primary; colour contrast is supplementary. Yellow TWSI acceptable in DEM spaces if profiling is the primary detection mechanism. Per VIS BPC Opus synthesis: "use detectable profile rather than relying on colour alone."
**Status:** RESOLVED-CONSENSUS (Opus 2026-03-29 VIS BPC synthesis)

### Tertiary conflict: vibrotactile floor panels vs TWSI
**Populations:** DBL (vibrotactile alerting) vs VIS/DBL (tactile wayfinding)
**Resolution:** RESOLVED-CONSENSUS. TWSI takes precedence as primary navigation aid. Vibrotactile alerting provided via wearable pager system.
**Status:** RESOLVED-CONSENSUS (Opus 2026-03-29 DBL BPC synthesis)
