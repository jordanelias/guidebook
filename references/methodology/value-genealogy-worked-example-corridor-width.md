Source dossier preserved verbatim at `sessions/session_2026-07-13-value-genealogy-mining-dossier.md` (auditable). Adversarially reviewed 2026-07-13; corrections applied (changelog at end). Everything below is mined from the project's own corpus (md + yaml + json + DB, session 2026-07-12); claims carry their corpus anchors. This document *demonstrates* the adjudication method on one parameter; it does not itself change any determination.
**Why this parameter:** corridor width is the canonical convergence-not-evidence case (`governance/tier-system.md` §3) — and, as this genealogy shows, it is also the clearest case of measurement-paradigm confusion (swept path vs turning circle), device-class transfer fallacy, multimodal function (signing space), and the project's own value drift (four coexisting "Guidebook values" for one item).

**The adjudication in one line:** 1800mm has more documents and zero traceable roots; 2440/2400mm has two independent roots plus an untraced regulatory echo pointing the same way. Precedent counts documents; evidence counts roots. The determination engine must therefore weight the independence count, never the citation count — which is exactly what the genealogy schema (companion DR) makes queryable.

## 2. The measurement-paradigm layer — why "turning radius" cannot adjudicate anything

The owner's directive: wheelchairs manoeuvre by **swept path**, not abstract turning radius. The corpus already holds the full argument (`manoeuvring-footprint-vs-turning-radius-methodology.md`, OPERATIVE, closing GAP-272 per owner directive "turning radius is disingenuous compared to swept path/turning manoeuvres"):

1. **No one pivots in place.** REF-00736 (Chaikhot et al. 2023, T1, VERIFIED): 97% execute spin turns — axis at the inner wheel, asymmetric footprint, outer wheel sweeping a ~1.4m arc; 3% roll turns. (Population-validity caveat the corpus itself records: n=10 able-bodied novices — external mining item.)
2. **The forward sweep is excluded from the circle.** Footrests project 250–400mm (manual) to 500mm+ (power with elevating legrests).
3. **Drive system changes the envelope shape**, not just its size: MWD ≈ pivot; RWD sweeps wide at the front; FWD fishtails rearward (BS 8300-2 annex; Steinfeld 2010).
4. **Manoeuvre type is the real variable:** 90° corridor entry (the median daily turn is 39°, IQR 24–67° — Sonenblum 2012) ≠ 180° in an enclosed bay (**2400mm** entire-sample) ≠ open-ended bay (**1925mm**, with the source's own caveat that open ends let users "lengthen out their turn" — and the corpus previously cited the smaller figure selectively: GAP-266) ≠ 360° ≠ transfer turns. Scooters are Ackermann-steered: **no pivot exists**; a multi-point envelope "geometrically different from a circle… currently absent from the corpus" (audit F-X4).

Consequence for adjudication: **values are commensurable only within a measurement paradigm.** A 1500mm static turning circle cannot corroborate *or* contradict a 2400mm enclosed-bay swept envelope — they measure different physical events. The genealogy schema therefore records `measurement_paradigm` per extracted value, and the independence count runs *within* paradigm. The corpus has already deprecated "turning circle/radius" as a synthesis parameter in favour of "manoeuvring footprint / swept envelope qualified by manoeuvre type + drive system" (methodology BPC §5).

**The dignity/injury stake, so this is not mistaken for pedantry** (methodology BPC §6): inner-hand braking force in a spin turn is 15.3±15.7× straight-line push; ~900 turns/day; manual-user upper-limb injury prevalence 55–72%. *"A 1525mm corridor isn't merely cramped; it forces the inferior turning manoeuvre repeatedly… for the user's entire wheelchair-using lifetime."* Corridor width is an **injury-dose parameter**, not a comfort parameter — which is also why fifty-year-old fit-cohort anthropometry (GAP-295: 2010 base now stale against heavier chairs and scooters) is an integrity problem, not a citation nicety.

## 3. The device-class layer — a value rooted in one class is not evidence about another

Corpus device ladder (`mobility-built-environment.md`; `bariatric-turning-radius-built-environment.md`; `body-sizes-supplementary-populations.md`): manual self-propelled (1500 floor / 1700 best-practice, + footplate sweep) · power by drive type (1675–1800+; ICC A117.1-2017 concedes 1702mm powered; chair width 700–800mm vs ADA's 915mm corridor minimum "set for manual wheelchair passage only") · scooter (1800+, multi-point envelope unmodelled) · bariatric (seat to 1220mm; 1900mm turning new-build; the 2400mm entire-sample figure suppressed in favour of 1925mm — GAP-266) · children's powered chairs (MEXT ≥2400mm) · **attendant-propelled: total corpus silence** (external mining item).

The class-transfer fallacy is already *named by the codes themselves*: PT DL 163/2006 and KR 편의증진법 explicitly state their 1500mm figures are manual-wheelchair-based; SG BCA 2025's revision is cited in-corpus as "explicit recognition that prior code insufficient for contemporary device footprints." Adjudication rule: `device_class` is recorded per extracted value; cross-class application is a *derivation* (recorded, down-weighted) never an *identity*.

## 4. The multimodal layer — what 2440mm is actually for

The same physical width serves functionally distinct programs, each with its own evidence stream and grain:

- **Signing conversation while walking (DEAF):** ≥2440mm primary / 1830mm secondary / 3050mm exterior; 900–3050mm interlocutor distance by group size; 3m unobstructed sightline depth; glazed junctions; ≥300 lux diffuse, no backlighting, matte mid-tone walls for skin-tone legibility (`deaf-spatial-design.md` L28–30; REF-00338/00342/00341). The corpus's own honesty markers are exemplary and must survive every rendering: the figure is **ASL-derived at one institution** ("an ASL-derived floor, not a universal value" — cross-language validity for BSL/DGS etc. unmined), "no independent replication of the quantified spatial dimensions exists," confidence LOW for dimensions / MODERATE for principles; and **no jurisdiction on earth specifies signing-space dimensions** (14-language Tier-6 absence) — regulatory silence on an entire language community's spatial needs is itself a genealogy finding.
- **Two-device passing + swept envelopes (MOB/SCI):** §§2–3 above.
- **Linked two-person travel:** sighted-guide pairs (VIS), intervenor pairs (DBL — ≥1500mm, THIN-BASE, "zero Tier 1–2 evidence"; Protactile contact-space is a global standards absence), companion walking (DEM).
- **Subsumption, not conflict** (CON-0122, HIGH): 2440 ⊃ 1800 — a designer specifying MOB-only silently deletes DEAF signing space. And the DB residue proves the invisibility mechanism: **DEAF is absent from E-08's `item_population_links`** (MOB/VIS/DEM/SCI only) even though DEAF holds the anchor evidence. Top-down population links, maintained by hand, dropped exactly the population whose evidence is strongest — the handshake's reverse-mapping exists to make that impossible.

## 6. The handshake, demonstrated on this cell

Top-down and bottom-up derivations already exist for this parameter and **have no common key** (FDR-BAB-04: ≥1800×1800 device envelope, keyed d465+device — defined in `references/fdr/accessible-bathroom-and-grab-bar.md`, scoped to entrance/door-approach turning, echoed in the circulation-geometry file; `references/fdr/deaf-spatial-design.md` FDR-DEAF-01: ≥1500 dyadic/≥2400 group signing, keyed d310+DEAF, flagged NOVEL — "absent from BPC"). Under the companion DR:

- (E-08 × MOB): population path (MOB BPC, codes) + functional path (d465 swept envelope, device classes) → `derivation_paths = dual` — convergence across *derivation paths* recorded as evidence, per mission doctrine #3's existing logic.
- (E-08 × DEAF): population path (DeafSpace Co-1/Co-2 + community claim) + functional path (d310/d330 communication mechanism — named, but the claim's normative force stays community-rooted) → `dual`, with the cultural-claim protection explicit: a population-only derivation would still be assertable; the handshake records *which* path carried it, it never demands mechanism where the claim is linguistic-cultural.
- The old CORRIDOR-W "conflict" resolves the way the project once resolved it by hand and then had no structure to remember: each side's **mechanism and physical variable** named → variables differ → no conflict; subsumption (CON-0122) recorded instead.
