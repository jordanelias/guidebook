## Batch 1: IE, NZ, SG

### Ireland (IE)
**Statutory:** TGD M 2010 (updated 2022)
- Ramp: preferred 1:20; max 1:12 with alternative means for wheelchair users beyond 2000mm total rise
- Landing: 1800×1800mm top/bottom; 1500mm intermediate
- Threshold: level entry preferred; ≤1:50 gradient = level
- Door: 1000mm effective clear width (entrance)
- Stair: rise 150–180mm; going 300–450mm; no open risers
- WC at entry level or storey containing main living room
**Co-1/Tier 2:** Irish Wheelchair Association Best Practice Access Guidelines — ramp 1:20 preferred, 1500mm min width (1800mm frequent use); turning circle 1800mm diameter; handrails 900–1000mm both sides
**Tier 5:** CEUD Building for Everyone (NDA) — comprehensive 10-booklet series; level access as primary design approach
**Notes:** TGD M 2022 represents significant update from 2010. IWA guidelines provide best-practice values that exceed statutory minimums. IE has strong Co-1 infrastructure through CEUD.

### New Zealand (NZ)
**Statutory:** NZS 4121:2001 / NZBC D1/AS1
- Ramp: max 1:12; landing max rise 750mm
- Threshold: preferably zero; max 20mm if unavoidable; max 56mm if 1:8 ramp both sides
- Door: 760mm min clear opening
- Corridor: 1200mm min; wider doorways compensate for narrow corridors
- Controls/signs: 1400–1700mm above floor
- Counter: accessible section required at service counters
**Co-1/Tier 2:** Disabled Persons Assembly NZ; CCS Disability Action; Blind Foundation NZ; BRANZ universal design bulletin 662
**Tier 5:** BRANZ Issue Bulletin 662 (2021) — universal access into homes; detailed level entry detailing (timber, concrete, decking thresholds)
**Notes:** NZS 4121:2001 does not apply to private residential dwellings by legislation but recommended. BRANZ provides detailed residential guidance. BCA Accessibility Code 2025 is a significant update — check for NZ equivalence.
**Key finding:** NZ control height 1400–1700mm is higher than many jurisdictions (vs ADA 380–1220mm, EN 17210 400–1100mm). Cross-reference with reach-range slug.

### Singapore (SG)
**Statutory:** BCA Accessibility Code 2019 (superseded by 2025 edition)
- Ramp: min 1200mm wide; kerb ramp max 1:10
- Controls: clause 9.2 (2025 edition consolidates operating devices)
- Anthropometric data: 2025 edition adds explicit reach range and clearance data
**Co-1/Tier 2:** SgEnable; SPD
**Tier 5:** BCA Universal Design Mark; AIC/CLC Dementia-Friendly Design Guide
**Notes:** BCA 2025 is a major update — adds anthropometric basis, relaxes button activation force, consolidates control device requirements. Updates SG from 2019 to 2025 standard.

## Batch 2: CH

### Switzerland (CH)
**Statutory:** SIA 500:2009 (Hindernisfreie Bauten / Constructions sans obstacles)
- Three categories: publicly accessible buildings, buildings with apartments, buildings with workplaces
- Flexibility margin: "conditionally permissible" and "preferably" allow graduated requirements
- Federal/cantonal/municipal layers — compliance determined by local jurisdiction
**Co-1/Tier 2:** Pro Infirmis; SZB (Schweizerischer Zentralverein für das Blindenwesen); SGB-FSS
**Tier 5:** SIA 500 commentary; Procap design guides
**Notes:** Swiss system is notable for graduated requirement levels — more nuanced than binary mandatory/advisory approach in most jurisdictions. Good model for three-tier hierarchy alignment.

## Batch 3: BE

### Belgium (BE)
**Statutory:** Three regulatory regimes — Flanders (Toegankelijkheidsverordening 2010), Brussels (COBAT Title IV), Wallonia (Walloon Code)
- Flanders: public buildings only since 2010; 2023 concept note approved for update/simplification
- Brussels: Regional Town Planning Regulation covers accessibility for persons with reduced mobility
- Wallonia: separate accessibility code
**Co-1/Tier 2:** GRIP; Fevlado; Ligue Braille
**Tier 5:** Inter Handboek Toegankelijkheid (Flanders); CAWaB guide (Wallonia)
**Notes:** Belgium's three-region system with separate regulations is unique. Flanders accessibility regulations apply only to public buildings — residential gap. AccessibleEU provides overview of Belgian framework.

## Batch 4: DK, FI

### Denmark (DK)
**Statutory:** BR18 (Bygningsreglementet 2018)
- Chapter on accessibility requires accessible routes to and in buildings
- DS 3028 provides detailed accessibility standards
- Regularly updated via amendments
**Co-1/Tier 2:** DH (Danske Handicaporganisationer); Danmarks Blindesamfund
**Tier 5:** SBi-anvisning 230 (BUILD/AAU research guidance)
**Notes:** BR18 is comprehensive but accessibility chapter values need direct extraction from bygningsreglementet.dk. Danish system integrates accessibility into general building code rather than separate standard.

### Finland (FI)
**Statutory:** Accessibility Decree 241/2017 (Valtioneuvoston asetus esteettömyydestä)
- Dedicated accessibility decree separate from building code
**Co-1/Tier 2:** Vammaisfoorumi; Näkövammaisliitto; Kuuloliitto
**Tier 5:** Invalidiliitto ESKEH; ARA (Housing Finance and Development Centre)
**Notes:** Finland's dedicated accessibility decree is among the most progressive Nordic approaches. Invalidiliitto (Finnish Association of People with Physical Disabilities) publishes ESKEH assessment tool — strong Co-1 source.
