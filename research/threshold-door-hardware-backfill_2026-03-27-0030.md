## 3. Cross-Jurisdiction Synthesis (preliminary — pre-Co-1)

### 3.1 Threshold height
| Best practice | Jurisdictions |
|---|---|
| 0 mm (Nullschwelle/Nullterskel) | DE, NO (Tier 5), JP |
| ≤5 mm | BR, AU (square edge) |
| ≤13 mm | US, CA, SG, ISO |
| ≤15 mm | UK, SE, CN |
| ≤20 mm | FR, NL, DK (Tier 5), FI, ES, PT, KR, EU, JP (fallback) |
| ≤25 mm | NO (Tier 6), DK (Tier 6), IT, CH |

**Best practice direction:** Zero threshold (DE/NO/JP). Any threshold >0 mm is a barrier for powered wheelchair/scooter users.

### 3.2 Clear opening width
| Bracket | Jurisdictions |
|---|---|
| ≥900 mm | DE (public), most Tier 5 recommendations |
| ≥850–860 mm | AU, CA (B651), FI, BE, NL, SG, NO, EU |
| ≥800–815 mm | US, UK (new), FR (new), ES, KR, CN, JP, BR, ISO, CA (NBC), DE (dwelling) |
| ≤780 mm | UK (existing), FR (existing), DK, PT, IT (visitabile) |

**Best practice direction:** ≥900 mm for comfortable powered wheelchair passage.

### 3.3 Handle height
| Range | Jurisdictions |
|---|---|
| 850–1050 mm | DE (tightest) |
| 900–1100 mm | AU, CA, BR, SG |
| 800–1100 mm | UK, SE, NO, DK, FI, NL, BE, EU, ISO |
| 800–1200+ mm | NZ, ES, KR, JP, US, FR, IT |

**Best practice direction:** 850–1100 mm (DE lower, AU upper).

### 3.4 Operating force
| Max force | Jurisdictions |
|---|---|
| ≤20 N | AU, UK (BS 8300), NO |
| ≤22 N | US, CA, PT, SG, ISO |
| ≤25 N | DE, NL, ES, BR, FI, SE, EU |
| ≤30 N | UK (Part M), CH, BE, KR, IT |
| ≤50 N | FR (outlier) |
| Not specified | JP, CN |

**Best practice direction:** ≤20 N (AU/UK BS 8300/NO).

### 3.5 Door closer sweep time
| Min time | Jurisdictions |
|---|---|
| ≥5 s | US, UK (BS 8300) |
| ≥3 s | UK (Part M), AU, CA, NZ, SG, ISO, SE |
| Not specified | Most EU/Nordic/Asian |

**Best practice direction:** ≥5 s.
