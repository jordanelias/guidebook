## Method
Each item's specification was matched to the BPC slug that is its **primary evidence source**, grounded in the 82-slug set (A.1-confirmed) and each slug's evidence depth (source_slug_links count). Category letters (A–K) do **not** map cleanly to topic areas — the item names show A=acoustics, B=lighting, C=colour/contrast, D=wayfinding, E=entrances/circulation, F=sensory/air/thermal, G=seating/bathroom/worksurface, H=controls/comms, I=hardware/kitchen/bathroom, K=deafblind/tactile — so mapping is per-item content judgment, not mechanical.

Confidence: **HIGH** = unambiguous 1:1; **MED** = defensible best-of-small-set (alternative recorded); **LOW** = genuinely split or no appropriate slug → deferred, not forced.
