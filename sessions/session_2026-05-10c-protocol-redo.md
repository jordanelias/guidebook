- **Directly challenged:** ZH (practitioner-directed SI model), KO (same), DA (McKee 2007: no evidence, possible harm)
- **Reframed:** NL + FI position low-stimulus environment as PRIMARY design strategy, with sensory rooms as SUPPLEMENTARY. This inverts the BPC framing.

- **Trademark concern:** Snoezelen is a registered trademark (FR Cairn review), creating commercial bias in evidence base

**BPC reviewer action needed:** The Opus synthesis should note cross-jurisdictional challenges. The claim holds within Unwin et al.'s framework but is not universally confirmed.
